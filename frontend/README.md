# Frontend

## 사용법

```
npm install
```

```
npm run serve
```

## 폴더구조

```bash
.root
├─node_modules
├─public
└─src
    ├─api           // 비동기 api
    ├─assets
    ├─components    // 컴포넌트
    ├─config        // 설정파일
    ├─router        // 라우터
    ├─store         // 상태 관리
    │  └─modules
    └─views         // 뷰 페이지
```
